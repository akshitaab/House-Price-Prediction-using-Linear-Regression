# 🏠 House Price Prediction App

This beginner-friendly project uses **Linear Regression** to predict house prices based on area, bedrooms, and bathrooms.

## 🔧 How It Works
- Input your house details
- The model predicts estimated price
- Built with: `Streamlit`, `pandas`, `sklearn`, `matplotlib`

## 🚀 Run It Online
Deploy this app for free using [Streamlit Cloud](https://streamlit.io/cloud).

## 📦 How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
