## 📌 Overview

| | |
|---|---|
| **Goal** | Predict house prices based on area, number of bedrooms, and bathrooms |
| **Model** | Linear Regression (Scikit-learn) |
| **Interface** | Interactive Streamlit web app |
| **Deployment** | Streamlit Cloud |

---

## 🔄 How It Works

```
User Inputs (Area, Bedrooms, Bathrooms)
            │
            ▼
     Data Preprocessing
            │
            ▼
    Linear Regression Model
            │
            ▼
     Predicted House Price 💰
```

## 🚀 Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/akshitaab/House-Price-Prediction.git
cd House-Price-Prediction
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Launch the app**
```bash
streamlit run app.py
```
